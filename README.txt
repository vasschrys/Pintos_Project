To test our project:

In /pintos/src/threads, remove the old build file:
rm -r build

In /pintos/src/threads, call the following to create the build file:
make

In /pintos/src/threads/build, call the following to run all tests:
make check

In /pintos/src/threads/build/tests/threads, call the following to see result of a test:
cat TESTNAME.result

(example: cat alarm-priority.result)
